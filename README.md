## Working on Colab

- Click the link below to the corresponding notebook
- Work on it
- Edit > Clear all outputs (or use the `./clean-notebooks.sh` script when working locally)
- File > Download .ipynb
- Replace the file in the repository
- `./clean-notebooks.sh notebooks/*`
- git add/commit/push
